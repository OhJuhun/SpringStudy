# Keywords
## Optional
    값을 발견하지 못한 경우 null return
    미사용시 [], '' return

## ResponseEntity
    HttpEntity를 상속 -> HttpHeader, Body를 가질 수 있다.
    HttpStatus와 함께 사용. 협업시 최적화된 모듈

# Annotations
## @Transactional
    트랜잭션화로 자동 RollBack이 가능하다.
    주로 modify, delete 등의 Query에 사용
